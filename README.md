# Projeto: Site Premium para Dentista Especialista

Este repositório contém as especificações e diretrizes para a criação de um site premium para um dentista especialista, focado em construir uma marca pessoal forte, elegante e memorável.

## Objetivo
Destacar o profissional como referência em sua área, transmitindo confiança, exclusividade, precisão, atendimento humanizado e excelência.

## Referências de Estilo
- Apple
- Leica
- Bang & Olufsen
- Aesop
- Porsche
- Estúdios de arquitetura contemporânea

## Identidade Visual
- Layout editorial
- Muito espaço em branco
- Tipografia sofisticada (Canela, Cormorant Garamond, Inter, Manrope)
- Transições suaves e design assimétrico

---

## Especificações Completas

Crie um site premium para um dentista especialista, focado em construir uma marca pessoal forte, elegante e memorável. O objetivo não é apresentar uma clínica, mas destacar o profissional como referência em sua área, transmitindo confiança, exclusividade, precisão, atendimento humanizado e excelência.

O projeto deve parecer desenvolvido por uma agência internacional de UX/UI especializada em marcas de luxo, com qualidade equivalente a um projeto de mais de R$ 50.000.

O design deve fugir completamente dos sites tradicionais de odontologia. A experiência precisa ser moderna, sofisticada e emocional, valorizando o profissional através de uma identidade visual única e uma navegação envolvente.

### PALETA DE CORES
- **Fundo principal:** Branco (#FFFFFF), Off White (#FAF8F5), Cinza muito claro (#F5F5F5)
- **Detalhes:** Bege Claro (#EDE3D3), Champagne (#DCC4A3)
- **Texto:** Grafite (#222222), Cinza Escuro (#3B3B3B)

### REQUISITOS OBRIGATÓRIOS
- Não utilizar emojis ou ícones infantis.
- Backgrounds exclusivos e profissionais.
- Imagens premium originais com aparência realista.
- Totalmente responsivo e otimizado para SEO.
- Acessibilidade WCAG AA e Dark Mode elegante.
